특정 상황(일상 대화, 공식 자리, 축하 자리 등)을 설정해두고 학습 중인 언어로 대화를 이어가다가 막히는 순간 모국어로 말하면 "이 상황에서는 이렇게 표현합니다"를 바로 알려주는 앱 컨셉입니다. 사전을 하나씩 찾는 번거로움을 없애고, 맥락에 맞는 자연스러운 표현을 즉시 제안한다는 점이 기존 AI 대화 앱과의 차별점으로 보입니다.

## 피드백 / 더 발전시킨다면
- **상황 난이도 시스템**: 일상 대화 → 공식 미팅 → 즉흥 상황처럼 단계별로 올라가는 커리큘럼을 짜면 학습 진척이 눈에 보이고 이탈률을 줄이는 데 도움이 될 가능성이 있습니다
- **모국어 개입 횟수 트래킹**: 같은 상황을 반복할 때 "지난번보다 모국어 전환을 2번 덜 했습니다"처럼 피드백하면 실력 향상을 체감할 수 있습니다
- **음성 인식 품질이 핵심 리스크**: 비표준 억양·속도 처리가 부실하면 학습 몰입이 즉시 깨지므로, STT 엔진 선택이 제품 완성도를 결정하는 핵심 변수일 것입니다
